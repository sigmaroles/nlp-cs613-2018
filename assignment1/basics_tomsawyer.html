Heap's law model fitting¶

The basic equation to be fit is
\begin{equation} |V| = k N^\beta \end{equation}
where $|V|$ denotes size of vocabulary and $N$ is size of corpus, $k$ and $\beta$ are parameters to be estimated.

To do this, first the equation is converted to \begin{equation} log (|V|) = log (k) + \beta log (N) \end{equation} This is essentially a linear equation of the form \begin{equation} y = mx + c \end{equation}
Which can be fitted using a standard linear regression solver like scikit-learn LinearRegression.
